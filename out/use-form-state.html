    Use Form State
    When Hooks Run

This page has an example implementation of a custom useFormStatehook to handle form submissions. You can choose to borrow this hook for your projects in lieu of a separate form library. You may find it beneficial to fully own the form management code in your project rather than use a library so it is fully extensible. You can find all the files needed to implement this hook in src/lib/hooks/use-form-state
