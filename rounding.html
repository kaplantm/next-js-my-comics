JS Float Rounding using toFixed is a little crazy.

Here we demonstrate this and include an alternative method.

Examples using native Number.prototype.toFixed() method:

12.655 12.65
92.655 92.66

What the heck? the rounding is inconsistent.

Examples using custom stableToFixed method:

12.655 12.66
92.655 12.66

The rounding is consistent.
